# CONFIGURATION PROJET CAPCUT

## Paramètres de base
- **Format** : 16:9 (1920x1080) pour YouTube
- **FPS** : 30 fps
- **Export** : 1080p minimum, 4K recommandé
- **Durée totale** : ~3h00 (15 vidéos séparées)

## Assets à préparer avant de commencer
1. **Avatar TLD** — PNG fond transparent (pour overlays)
2. **Avatar TLD animé** — Version avec effets d'énergie (intro/outro)
3. **Logo TLD** — Version simplifiée pour le coin de l'écran
4. **Musique de fond** — Lo-fi / Tech ambient (bibliothèque CapCut)
5. **Son d'intro** — Whoosh + bass drop (3 secondes)
6. **Captures d'écran ChatGPT** — Filmer en live ou screenshots
7. **Police** : Space Grotesk ou Montserrat (cohérence avec le site)

## Palette de couleurs (identique au site)
- Fond sombre : `#0a0a0f`
- Cyan accent : `#00f5ff`
- Magenta : `#ff00aa`
- Jaune : `#f5ff00`
- Orange : `#ff9500`
- Vert : `#00ff88`

# STRUCTURE TYPE D'UN MODULE (Template)

Chaque module suit cette structure dans CapCut :

```
[0:00 - 0:08]  INTRO ANIMÉE — Avatar TLD + titre du module
[0:08 - 0:15]  TRANSITION — Accroche texte à l'écran
[0:15 - X:XX]  CONTENU — Alternance voix off / prompts à l'écran
[X:XX - X:XX]  RÉCAP — Points clés en bullet points
[X:XX - fin]   OUTRO — Avatar TLD + teaser module suivant
```

### Pistes CapCut (layers) :
```
Piste 1 (haut)   → Texte / Titres / Sous-titres
Piste 2          → Avatar TLD overlay (coin bas-droit)
Piste 3          → Captures d'écran ChatGPT / Visuels
Piste 4          → Background (fond sombre animé)
Piste 5 (audio)  → Voix off (TTS ou enregistrement)
Piste 6 (audio)  → Musique de fond (volume 15-20%)
Piste 7 (audio)  → Effets sonores (transitions)
```

# CONSEILS CAPCUT SPÉCIFIQUES

## Voix IA dans CapCut
1. Ajoute un **Texte** sur la timeline
2. Colle le script teleprompter
3. Clique droit → **Text-to-Speech**
4. Choisis voix FR masculine
5. Ajuste la vitesse (1.0x à 1.1x recommandé)

## Avatar en overlay
1. Importe l'avatar PNG (fond transparent)
2. Place sur la piste au-dessus du contenu
3. Redimensionne à ~15% de l'écran
4. Position : coin bas-droit
5. Ajoute animation "Fade in" à l'entrée
6. Ajoute animation "Fade out" à la sortie

## Texte des prompts
1. Utilise la police **Space Grotesk** ou **Courier** (style code)
2. Fond semi-transparent noir derrière le texte
3. Couleur texte : blanc ou cyan
4. Animation : "Typewriter" (machine à écrire)
5. Taille : assez grand pour être lu sur mobile

## Sous-titres automatiques
1. Menu **Texte** → **Sous-titres automatiques**
2. Langue : Français
3. Style : fond noir semi-transparent, texte blanc
4. Position : bas de l'écran

## Transitions recommandées
- Entre sections : **Glitch** ou **Zoom blur** (0.5s)
- Entre prompts : **Slide left** (0.3s)
- Intro/Outro : **Fade to black** (1s)
