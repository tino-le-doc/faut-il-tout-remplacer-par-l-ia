🔒 Politique de Confidentialité

Dernière mise à jour : Janvier 2026

📋

Introduction

Bienvenue sur TLD. Nous accordons une grande importance à la protection de vos données personnelles et à votre vie privée.

Cette politique de confidentialité explique quelles données nous collectons, pourquoi nous les collectons, comment nous les utilisons et quels sont vos droits concernant ces données.

🛡️ Notre engagement : Vos données ne sont jamais vendues à des tiers. Nous collectons uniquement les données nécessaires au fonctionnement du site.

👤

Responsable du traitement

Le responsable du traitement des données personnelles est :

  • Nom : TLD
  • Site web : https://martialfabrice1976-lang.github.io/Tino-le-Doc/
  • Contact : Via le tchat du site ou les réseaux sociaux

Données collectées

Nous collectons différentes catégories de données selon votre utilisation du site :

1. Données de compte (si vous créez un compte)

Donnée Finalité Durée de conservation
Adresse email Identification, récupération de mot de passe Jusqu'à suppression du compte
Pseudo Identification dans le tchat et la messagerie Jusqu'à suppression du compte
Mot de passe Sécurité du compte (stocké chiffré) Jusqu'à suppression du compte
Date d'inscription Statistiques internes Jusqu'à suppression du compte

2. Données de tchat et messagerie

Donnée Finalité Durée de conservation
Messages publics Fonctionnement du tchat public 30 jours maximum
Messages privés Messagerie entre utilisateurs Jusqu'à suppression par l'utilisateur
Liste de contacts Gestion de vos amis Jusqu'à suppression du compte

3. Données de navigation (anonymes)

Donnée Finalité Durée de conservation
Compteur de visites Statistiques du site Indéfiniment (anonyme)
Préférences (localStorage) Personnalisation de l'expérience Stocké localement sur votre appareil

⚠️ Note : Nous ne collectons PAS d'adresse IP, de données de géolocalisation, ni d'informations sur votre appareil au-delà de ce qui est strictement nécessaire.

⚖️

Base légale du traitement

Nous traitons vos données sur les bases légales suivantes :

  • Consentement : Lorsque vous créez un compte ou postez un message
  • Exécution d'un contrat : Pour vous fournir les services du site (tchat, messagerie)
  • Intérêt légitime : Pour améliorer nos services et assurer la sécurité
🔗

Services tiers

Notre site utilise les services tiers suivants :

Firebase (Google)

Nous utilisons Firebase pour l'authentification et la base de données temps réel. Firebase est soumis à la politique de confidentialité de Google.

  • Politique de confidentialité Firebase : firebase.google.com/support/privacy

GitHub Pages

Notre site est hébergé sur GitHub Pages. GitHub peut collecter des données techniques de base pour le fonctionnement du service.

  • Politique de confidentialité GitHub : docs.github.com/privacy

Tenor (GIFs)

Les GIFs partagés dans le tchat proviennent de Tenor (Google). L'intégration de GIFs est soumise aux conditions de Tenor.

Sécurité des données

Nous mettons en œuvre les mesures suivantes pour protéger vos données :

  • 🔐 Chiffrement : Les mots de passe sont chiffrés (Firebase Auth)
  • 🔒 HTTPS : Toutes les communications sont sécurisées via SSL/TLS
  • 🛡️ Règles Firebase : Accès aux données restreint et validé
  • 🚫 Anti-spam : Rate limiting sur les messages
  • ⚠️ Modération : Filtrage automatique du contenu inapproprié
🍪

Cookies et stockage local

Notre site utilise :

LocalStorage (stockage local)

Nous utilisons le localStorage de votre navigateur pour :

  • Mémoriser votre pseudo si vous êtes connecté
  • Compter les visites uniques (une par jour)
  • Stocker vos préférences d'affichage

Ces données restent sur votre appareil et ne sont pas envoyées à nos serveurs.

Cookies Firebase

Firebase peut utiliser des cookies techniques pour l'authentification. Ces cookies sont strictement nécessaires au fonctionnement du service.

🚫 Pas de cookies publicitaires : Nous n'utilisons AUCUN cookie de tracking, d'analyse comportementale ou de publicité.

👶

Protection des mineurs

Notre site peut être utilisé par des personnes de tous âges. Cependant, nous recommandons aux mineurs de moins de 13 ans de ne pas créer de compte sans l'autorisation de leurs parents ou tuteurs légaux.

Si vous êtes parent et que vous pensez que votre enfant nous a fourni des données personnelles, contactez-nous pour que nous puissions les supprimer.

📝

Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps en temps. La date de "dernière mise à jour" en haut de cette page indique quand la politique a été révisée pour la dernière fois.

En cas de modifications importantes, nous vous en informerons via une notification sur le site.
